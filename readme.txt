A WordPress plugin that allows you to fetch related comments for a post from other services eg: YouTube, 500px, Dribbble

== Description ==

A WordPress plugin that allows you to fetch related comments for a post from other services eg: YouTube, 500px, Dribbble

== Installation ==

The plugin is simple to install:

1. Download `related-service-comments.zip`
1. Unzip
1. Upload `plugin-template` directory to your `/wp-content/plugins` directory
1. Go to the plugin management page and enable the plugin
